Rust is a language that I approached because of its increasing popularity.

I can't say that I am an expert programmer, but i bought a manual and tried to write some example programs in it.
Its compiler is absolutely amazing. Coming from languages that uses Garbage Collectors to guarantee memory safety,
its ability to manage the memory efficiently without introducing GC was something that caught my attention.
At the time I was amazed by the concepts of the language and the instruments it uses to ensure memory safety.
The concepts of ownership and lifetime are complicate to relate to, but at the same time are an innovation that amazed me.
The compiler can caught the majority of the errors and gives you explanotory error messages. Really awesome.

In the future I hope that I can work on some projects using Rust because I think that it really is amazing.
