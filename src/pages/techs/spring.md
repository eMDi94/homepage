As I said for the Application Servers, I think that it may be too hard to configure and to maintain its configuration. Is it really worth the effort?

So, whay do you have the like? Did you copy&paste it? No, I rediscovered my love for it with Spring Boot.

I think that if a lot of people use it in a certain way, why would I go out of the way? So the Spring Boot project, with its Convention over Configuration, really helped me a lot.
Now I can use the full power of the Spring Framework overriding only the configuration that I need. Using Spring Boot made me a Java developer again. That's all I have to say.
