Java is the main language that I use regularly at work.
It was the first OOP language that I learned back in my third year of University.

Even though I use it a lot, I don't feel like saying that it one of my favourite languages. I know that it is one of the most popular languages, but I don't feel myself very comfortable writing code in it. Sometimes I think that its syntax and constructs are way to complicate to achieve some tasks.

Sometimes the **NullPointerExceptions** really get on my nerves. I think that there are better ways of handling the absence of information, but this is how it is done in the language, and there is nothing we can do about it.

When I started working at my current company, I started to learn also the the Enterprise Edition using JBoss and Wildfly. During the years I also tried Quarkus and Spring Boot. But this is a story for another post.
